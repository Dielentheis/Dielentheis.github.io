Projects

Plont

Plont is a web application that aims to bring gardening into the technical age. With the aid of Plont, you can optimize the usage of your garden's space based on your desired plants' needs and be notified of when it's time to plant, and harvest, and water your garden based on your zip code. It's built using PostgreSQL, Sequelize, JavaScript, Express, and Angular. source code

Don't Tell Me

Don't Tell Me is a 2-day hackathon project and Chrome extension that blocks Game of Thrones spoilers from the user's Facebook NewsFeed. source code

Mood Lighting

Hackathon project for Developer Camp 2016. Won "Spotlight" Award. Changes the color of the lights depending on the emotional content of the messages. The saturation is determined by the strength of the emotions in each messages (i.e. neutral messages produce a less vibrant hue). Used Philips Hue Lights, Watson Alchemy Language API, and PubNub for messaging. source code

R.A.M. (Realistic Artificial Memories)

R.A.M. is a faux e-commerce site selling memories. It's built using PostgreSQL, Sequelize, JavaScript, Express, and Angular. source code

Well, This is Awkward

Well, This is Awkward is a hardware hackathon project using Tessel 2 and JavaScript. It monitors a conversation between 2 people and, when it detects an awkward silence, it plays cricket noises and sends a text message to the user with a conversation starter.
